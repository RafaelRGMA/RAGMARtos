<h1>RAGMARtos - Sistema Operacional de Tempo Real Cooperativo</h1>

<blockquote>
  <strong>Atenção!!!</strong> Este Sistema Operacional é um Soft Real Time, ou seja, nunca deverá ser utilizado em projetos críticos e Hard Real Time.
